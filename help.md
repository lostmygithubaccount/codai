I am codai, a coding AI assistant.

Commands:

- /ls: List files in the current directory.
- /cd {directory}: Change the current directory to the specified directory.
- /tree or /t: Display the directory structure of the current directory in a tree-like formatk.
- /read {filename}: Read a file.
- /write {filename}: Write to a file (.md or .py).
- /image: Generate an image summary of the last message.
- /wget {url}: Download a file from the specified URL.
- /rename {old_name} {new_name}: Rename a file from 'old_name' to 'new_name'.
- /exit, /quit, or /q: Exit the program.

To execute a command, type it into the chat and press enter. Chat away.
